FBMeet

Application made in Facebook Hackathon (2nd place). It's a Chrome extension that integrates FB chat with FB events trying to make it easier for the user to create quick events and invite his friends through a minimal UI and a simple Drag'n Drop feature.

Link to .crx: http://fbmeet.herokuapp.com/FBMeet.crx
